# About

This is the website for the Portland Accessibility and User Experience Meetup (PDX A11Y UX). The current version can be seen at [pdxa11yux.org](https://pdxa11yux.org).

This site is built using [Jekyll](https://jekyllrb.com/), in addition to plain HTML and CSS. The [source code](https://github.com/pdxa11yux/pdxa11yux.org) is available on GitHub.
