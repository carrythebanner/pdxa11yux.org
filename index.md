# Portland Accessibility and User Experience Meetup (PDX A11Y UX)

## About

The Portland Accessibility and User Experience (<abbr>PDX A11Y UX</abbr>) Meetup addresses how to incorporate accessibility and universal design principles into project planning, user research, and UX evaluation. We are based in the [Portland, Oregon](https://en.wikipedia.org/wiki/Portland,_Oregon){: rel="external"} metro area.
